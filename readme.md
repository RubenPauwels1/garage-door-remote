# Garage Door Remote

A garage door remote API for Raspberry Pi using WiringPi.

## Technology used

  - NodeJS
  - Express
  - Firebase
  - WiringPi

## Installation and usage
The app needs admin rights to run.
Do not forget to create an .env file.

```sh
$ npm install
$ sudo npm start
```
or
```sh
$ yarn install
$ sudo yarn start
```
